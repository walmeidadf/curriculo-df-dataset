# Currículo em Movimento do DF — Interface de Busca

Interface de busca para o dataset público do *Currículo em Movimento do Distrito Federal – Ensino Fundamental* (2ª edição, SEEDF).

## Como usar

1. Digite uma palavra-chave (ex: `frações`, `oralidade`, `gêneros textuais`)
2. Use os filtros para refinar por componente curricular, ciclo, bloco ou ano escolar
3. Os resultados atualizam automaticamente ao mudar os filtros
4. Clique em **Exportar seleção como CSV** para baixar os registros filtrados

## Sobre o dataset

- **318 registros** de objetivos e conteúdos curriculares
- **10 componentes curriculares**: Arte, Ciências, Ed. Física, Ed. Religiosa, Ensino Religioso, Física, Geog., Hist., L. Portuguesa, Matemática, Química
- **2 ciclos**: 2º Ciclo (Anos Iniciais, 1º–5º ano) e 3º Ciclo (Anos Finais, 6º–9º ano)
- Dataset completo: [walmeidadf/curriculo-ensinofundamental-df](https://huggingface.co/datasets/walmeidadf/curriculo-ensinofundamental-df)

## Fonte

Currículo em Movimento do Distrito Federal – Ensino Fundamental, 2ª edição, SEEDF.
